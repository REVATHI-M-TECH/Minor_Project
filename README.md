# Personalized Lung Risk Prediction System

## Project Description
A machine learning–based system to predict lung disease risk using clinical and lifestyle data, with future integration of CT scan–based features.

## Project Structure
- data/raw: Original dataset
- data/processed: Cleaned dataset
- notebooks: Jupyter notebooks for analysis, cleaning, and modeling
- src: Reusable preprocessing scripts

## Workflow
1. Dataset understanding
2. Data cleaning and preprocessing
3. Feature encoding and scaling
4. Baseline model training

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Jupyter Notebook

## Team Roles
- Clinical data analysis and risk factor modeling
- CT scan feature extraction (CNN – future scope)

## Future Scope
- Deep learning on CT scan images
- Multimodal learning
- Real-time clinical decision support
